# ElevenLabs Integration Setup

To enable the high-quality AI voice, you need to add your ElevenLabs API Key.

1.  Create a file named `.env.local` in the root folder: `c:\Users\manna\Documents\Resume2Interview\resume2interview\`
2.  Add your API key to it:

```env
ELEVENLABS_API_KEY=your_api_key_here
```

**Note:** If you don't have a key, the app will automatically fall back to the standard browser voice (Web Speech API).
